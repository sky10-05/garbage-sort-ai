# 自治体対応型AIごみ分別支援システム

## 概要

自治体対応型AIごみ分別支援システムは、カメラで撮影したごみをAI画像認識で判定し、自治体ごとの分別ルールに基づいて適切な捨て方を案内するWebアプリケーションです。

画面表示だけでなく音声案内にも対応し、利用者がごみの分別方法に迷わず確認できることを目指します。

## 開発背景

ごみの分別ルールは自治体ごとに異なります。そのため、引っ越してきた住民、高齢者、外国人住民などが正しい分別方法を判断しづらい場合があります。

特に、乾電池、スプレー缶、モバイルバッテリーなどは、誤った捨て方をすると火災や事故につながる可能性があります。

本システムでは、AI画像認識と自治体別の分別ルールDBを組み合わせることで、利用者が安全かつ正しくごみを分別できるよう支援します。

## YOLO推論の試験実行

試験学習済みモデルを使う場合は、以下のようにモデルパスを指定して起動します。

```bash
YOLO_MODEL_PATH=runs/detect/train-11/weights/best.pt python3 app.py
```

YOLOモデルを使わず、従来の仮判定結果で画面遷移だけ確認する場合は、以下を使用します。

```bash
USE_MOCK_INFERENCE=1 python3 app.py
```
